# Elo-Design-Challenge
This repository contains the design files and documentation for the Parallel Linkage Neck Mechanism developed for the Elo Robotics Technical Interview Challenge. The project focuses on designing a compact 3 DOF robotic neck mechanism using a parallel linkage system adapted from the Pollen Robotics Amazing Hand finger mechanism.

The repository includes the CAD files, STL files for 3D printing, and supporting documentation. The design integrates three STS3215 servo motors, a 1:1 base transmission, a parallel linkage mechanism for pitch and tilt, and a slip ring for continuous rotation. All detailed documentation and design explanations are provided in the Neck Design Mechanism PDF included in this repository. This document contains:

  * Bill of Materials (BOM) 
  * Assembly Instructions 
  * Design Rationale
  * Parallel Linkage Kinematics
  * Tolerance Decisions 
  * 3D Print Settings 

The Neck Design Mechanism PDF should be referred to for a complete overview of the design process, mechanism analysis, and assembly details.
